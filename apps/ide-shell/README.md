# ide-shell

Static protocol-driven reference client for FWP.

Open `src/index.html` in a browser or serve the `src/` directory. The shell can
load fixture transcripts from `packages/formal-protocol/examples/ui-fixtures.json`.

This app is a seam-validation surface only. In the target stack, the canonical
IDE or product shell lives above `formal-claim`, not inside FWP.
